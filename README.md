# Eduly Landing Page

Simple landing page for [Eduly](https://github.com/eduly-ai/eduly) — a Manim coding agent that transforms papers into animated educational videos.

## Structure

```
├── index.html      # Single-page site
├── css/styles.css  # Minimal styles
├── js/main.js      # Theme toggle + smooth scroll
└── assets/         # Favicon
```

## Development

Just open `index.html` in a browser. No build step needed.

## Deployment

Hosted on GitHub Pages at [landing.eduly.ai](https://landing.eduly.ai).
