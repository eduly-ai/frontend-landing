Quick Example

Process a paper into animated explainers:

from eduly import EdulyBreakdownClient, EdulyAnimationClient

# Break down a PDF into atomic topics
breakdown, _ = breakdown_client.breakdown(
    file_path=Path("./paper.pdf"),
    model="gemini-2.5-flash-preview"
)

# Generate storyboards for each topic
for topic in breakdown.topics:
    storyboard, _ = breakdown_client.storyboard(topic=topic)

# Animate with the Manim agent
result = animation_client.animate_single(
    breakdown=breakdown,
    storyboard=storyboard,
    max_iterations=5
)

print(f"Video saved to: {result.video_path}")

How It Works

Eduly uses a three-stage pipeline to turn your educational content into animated explainers:

1

Document Breakdown

Extract atomic topics ‚Äî self-contained concepts that can stand alone as ~5 minute videos. Uses Gemini with Google Search to generate comprehensive explanations with modern context.

2

Storyboard Generation

For each topic, generate a visual storyboard in the style of 3Blue1Brown ‚Äî defining core visual metaphors, 8-15 scenes, and narration optimized for text-to-speech.

3

Manim Animation Agent

An iterative coding agent that researches Manim docs, generates Scene code, renders animations, and self-corrects on errors ‚Äî outputting vertical 1080√ó1920 MP4 videos.
